# tailwind css의 크기 단위 값
- css에서 사용하는 크기 단위 값 : px, em, rem, % 등
- rem : r=root. body 또는 html에 설정되어 있는 폰트 사이즈를 기준으로 함.
- em : 상대적인 값. 부모 요소와 관계가 있음. e.g. p(자식 요소)태그의 부모 요소인 div의 폰트 사이즈 20일 때, ptag의 폰트 사이즈를 2em으로 준다면 부모 요소x2=40
- px : 고정되어 있는 값. 브라우저 사이즈가 늘어나든 줄어들든 변화가 없음.
- html 또는 body의 폰트 사이즈가 16px이면 1rem은 16px이고 html 또는 body 폰트 사이즈가 20px이면 1rem은 20px.
- w : width
  - 비율일 경우, w-분할비율(w-
- m : margin
- p : padding
- 속성명 첫 글자-[단위] e.g. w-[23px]은 width:23px이라는 것과 동일한 표현
- e.g. w-[50%]는 w-1/2 즉 0.5 rem과 동일한 표현
- width full은 w-[100%]과 동일.
- 1rem=16px, 0.75rem=12px, 다른 분
