# 로그인 화면 반응형 레이아웃 만들기
1. 텍스트 input field 추가 ("매일의 기록 / 멤버가 되어주세요!)
   - 기본 설정은 auto width -> 우측 design 패널에서 field 왼쪽 정렬 -> field+shift 누른 채로 움직여서 왼/오 여백 40으로 통일 -> field 오른쪽으로 늘리면 auto width에서 auto height로 변경 (줄바꿈 되면 너비는 그대로 높이 값 변경) -> command+shift를 통해 오른쪽 여백도 40   
2. btn 추가 (이메일로 가입하기, 구글 계정으로 가입)
   - "이메일로 가입하기" 텍스트 추가 -> AL 추가(shift+A) -> fill을 통해 컬러 추가 -> 버튼 더블 클릭하면 303x48 사이즈(현재 40) -> AL 패널 우측 끝 아이콘 통해 위 아래 여백 4씩 추가 -> R값 추가 (둥근 모서리)
