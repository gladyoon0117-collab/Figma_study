# 03_콘스트레인츠 (Constraints)

## 1. 콘스트레인츠의 정의와 역할
상위 프레임의 영역 조정 시 하위 콘텐츠가 반응하는 방식을 정의하는 지능형 제약 조건 기능.

- **핵심 기능**: 하위 콘텐츠가 상위 프레임을 기준으로 어느 위치에 고정되고(Position. e.g. 상위 프레임 기준 상단 100px/좌측 100px에 위치), 크기는 어떻게 가변하는지(Resizing) 제어.
- **여백 설계**: 안쪽 여백(**Padding**)과 바깥쪽 여백(**Margin**)의 역할을 수행하여 디자인의 일관성 유지.
  - **패딩 (Padding)**: 상자 '안쪽'의 여백 (내용물이 벽에 붙지 않게 보호).
  - **마진 (Margin)**: 상자 '바깥쪽'의 여백 (옆 상자와 부딪히지 않게 거리 유지).
- **적용 조건**: 프레임(Frame)에 포함된 기능이라 내부에 위치한 콘텐츠에만 활성화됨. (오토 레이아웃/그룹 내 콘텐츠에는 적용 불가)
- **위치**: Position > X,Y 우측의 사각 아이콘 클릭.

## 2. 시각화 및 설정 (Indicator)
- **인디케이터 (Indicator)**: 레이어 선택 시 해당 레이어와 상위 프레임 경계선 사이에 나타나는 **파란색 점선**. (`Cmd` + `Shift` + `H`로 활성화) 현재 설정된 제약 조건의 방향을 시각적으로 보여줌.
  - e.g. 약 조건을 'Left'에서 'Right'로 변경하면, 파란색 점선의 위치도 왼쪽 벽면에서 오른쪽 벽면으로 옮겨감.
- **콘스트레인츠 설정 방식**: 우측 디자인 패널의 사각 아이콘을 클릭하거나 드롭다운을 통해 수평/수직 조건을 각각 설정. 활성화된 제약 조건 값은 패널에서 파란색으로 강조됨.

## 3. 상세 조건 설정 (Horizontal / Vertical)

### 콘스트레인트의 기본 작동 원리 (수평/수직)
- 피그마에서 요소를 생성하면 기본적으로 **좌측(Left)** 및 **상단(Top)** 콘스트레인트가 적용. 
  - **기본값의 의미**: "이 요소는 왼쪽 벽면을 기준으로 100px, 상단 벽면을 기준으로 100px 떨어져 있다"는 정의.
  - **고정 수치 유지**: 상위 프레임의 크기가 어떻게 변하더라도, 설정된 **여백 값(예: 100px)은 무조건 유지**.
  - **결과**: 상위 프레임을 늘리거나 줄여도 하위 콘텐츠는 처음에 정렬해둔 '왼쪽 위' 지점에 딱 붙어서 따라감.

### 수평(Horizontal) 설정
| 설정 값 | 동작 방식 | 특징 |
| :--- | :--- | :--- |
| **Left / Right** | 왼쪽 또는 오른쪽 벽면 기준 여백 수치를 무조건 유지하며 따라감. | 위치 고정 |
| **Left & Right** | 양옆 여백 수치를 고정하는 대신, **하위 콘텐츠의 너비**가 가변함. | **크기 가변** |
| **Center** | 상위 프레임의 중앙 지점을 기준으로 하위 콘텐츠의 위치 유지. | 중앙 정렬 |
| **Scale** | 상위 프레임 크기 변화 비율에 맞춰 콘텐츠 크기와 여백이 **정비율**로 대응. | 비율 가변 |

### Scale (비율 대응)
상위-하위 관계로 엮여 있는 콘텐츠를 대상으로 하며, 절대적 수치가 아닌 **부모 대비 비율**을 유지.

- **동작 예시**: 
  - 상위 프레임 Width가 2배(400px -> 800px)가 되면, 
  - 하위 콘텐츠의 크기도 2배로 커지고, 
  - 기존에 100px이었던 여백(Margin)도 비율에 맞춰 200px로 늘어남.

### 수직(Vertical) 설정
- **Top / Bottom**: 상단 또는 하단 기준 위치와 여백 유지.
- **Top & Bottom**: 상하 여백을 고정하고 콘텐츠의 높이를 가변시킴.

## 4. 실무 활용 및 실습 팁

### 반응형 상단 상태바(Status Bar) 설계 예시
- **전체 컨테이너**: `Left & Right` (해상도 변화에 따라 전체 너비 가변)
- **시간 레이어**: `Left` (왼쪽 고정)
- **배터리/와이파이 아이콘**: `Right` (오른쪽 고정)
- **중앙 시계/로고**: `Center` (중앙 유지)

### 주의사항: Center vs Left & Right
- **Center**: 중앙 위치만 고수하며 콘텐츠 크기는 변하지 않음.
- **Left & Right**: 양옆의 여백(Margin)을 사수해야 하므로, 프레임이 넓어지면 콘텐츠의 크기도 함께 커짐.
