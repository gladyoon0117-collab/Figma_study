# 06_배리어블 (Variables)

## 1. 배리어블의 정의 및 특징
- 디자인 속성(색상, 수치, 문자 등)을 **재사용 가능한 변수**로 저장하여 디자인 시스템과 프로토타입에 적용하는 기능. (피그마에서 **변수=value**. 예: bg, padding, label)
- 배리어블의 배리어블, 그룹, 모드, 콜렉션 조합으로 원하는 디자인 시스템이나 환경 구축 가능.

### Style vs Variables 비교
| 구분 | 스타일 (Style) | 배리어블 (Variables) |
| :--- | :--- | :--- |
| **특징** | 여러 값의 조합 (예: 그라데이션, 그림자 포함된 Fill) | 단일 값 (Single Value) |
| **확장성** | 다른 스타일이나 변수에 참조 불가 | **다른 스타일이나 변수의 값으로 사용 가능** |
| **활용** | 복잡한 그래픽 효과 정의 | **디자인 토큰(Token)** 구현, 라이트/다크 모드 전환 |

---

## 2. 배리어블의 4가지 종류

1. **Color**: Hex 값을 사용하며 라이트/다크 모드 등 테마 작업에 유용.
2. **Number**: 정수 및 소수점 숫자를 사용하며, 객체의 너비/높이, 폰트 크기, 간격(Gap), 투명도 등에 적용. 반응형 디자인 구축에 필수적.
3. **String**: 문자열 값을 사용하며, 다국어 지원이나 텍스트 내용 변경, 상태 전환 등에 활용.
4. **Boolean**: 참(True)과 거짓(False) 값을 사용하여 특정 상황에 따라 레이어를 숨기거나 표시할 때 활용. 레이어 속성이나 Component variant property가 True/False 값 갖고 있을 때 연결 가능.

## 3. 배리어블 등록 및 관리 (Interface)
좌측 사이드 바 Variables 클릭 > Local Variables 창 오픈

### Local Variables 창 구성
* **Collection**: 배리어블의 최상위 묶음. 좌측 상단에서 이름 수정 및 새 콜렉션 추가가 가능.
* **Groups**: 배리어블 이름 앞에 `그룹명/`을 붙이거나 선택 후 우클릭 `New group with selection`으로 그룹화 가능.
* **Mode (모드)**: 컬렉션 우측의 `+` 버튼으로 모드를 추가하여 데스크톱/모바일, 라이트/다크 모드 환경을 구축.
* **Create Variable**: 배리어블 생성. 배리어블 종류(number, color...) 선택해 선택된 레이어에 배리어블 추가.

### 세부 설정 (Edit Variable)
배리어블 우클릭 → 세부 설정 편집을 통해 다음 항목을 관리.
* **Scoping**: 해당 배리어블이 노출될 속성 범위를 제한 (예: 특정 수치는 Padding에만 보이게 설정).
* **Code Syntax**: 개발자와의 협업을 위해 코드 구문을 추가.
* **Hide from publishing**: 라이브러리 배포 시 해당 배리어블 제외 가능.

## 4. [실습] 디바이스 대응 콜렉션 구축 및 연결

### 1단계: 배리어블 등록 (Number 활용)
해상도 및 간격 제어를 위해 `Device` 콜렉션을 생성하고 다음과 같이 `Number` 배리어블 등록. (그룹명/이름 형식)

| 그룹/이름 | Desktop (Mode 1) | Mobile (Mode 2) | 활용처 |
| :--- | :--- | :--- | :--- |
| **resolution/width** | 1920 | 360 | 프레임 너비 |
| **height** | 1080 | 800 | 프레임 높이 |
| **padding-h** | 40 | 20 | 가로 패딩 |
| **padding-w** | 22 | 18 | 세로 패딩 |
| **spacing-lg** | 60 | 24 | 큰 간격 |
| **spacing-md** | 36 | 10 | 큰 간격 |

### 2단계: 배리어블 연결 (Apply Variable)
* **프레임**: 적용할 레이어 선택하고 너비(W)와 높이(H) 입력창의 아이콘을 클릭하여 `resolution` 변수 연결.
* **오토레이아웃**: Padding 및 Gap 수치 옆의 아이콘을 클릭하여 `spacing`, `padding` 변수 연결.

### 3단계: 모드 변경 및 확인 (Mode Switching)
설정된 배리어블은 우측 패널에서 클릭으로 전체 레이아웃 전환 가능.
* **루트**: 부모 프레임 선택 → 우측 **Layer 패널** → **Change Variable Mode** 아이콘 클릭 → 생성한 콜렉션(Device) → 모드 선택 (Desktop ↔ Mobile).
