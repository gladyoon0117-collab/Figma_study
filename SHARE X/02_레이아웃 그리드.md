# 02_레이아웃 그리드 (Layout Grid)

## 1. 레이아웃 그리드
디자인의 일관성과 사용성을 유지하기 위해 요소들을 줄이나 격자(Grid)에 맞추는 설계 도구. 피그마에서는 **Layout Grid** 기능을 통해 논리적인 그리드 시스템을 구현함.

- **목적**: 해상도 대응(반응형), 여러 명과의 협업 시 일관된 디자인 유지, 시각적 안정성 확보.
- **적용 대상**: **프레임(Frame)**에만 적용 가능 (오브젝트나 그룹에는 설정 불가).
- **제어**: 
  - `Shift` + `G`: 그리드 노출/비활성 제어.
  - Layout Guide 패널 내 `+` 버튼으로 추가, `-` 버튼으로 제거.
- 피그마 버전 업데이트에 의해 Layout Grid -> Layout Guide 명칭 변경

## 2. 그리드 시스템의 구성 요소
반응형 레이아웃을 잡을 때 다양한 플랫폼에 대응 가능한 그리드 시스템으로 조합하기 위해 가장 먼저 정의해야 하는 3가지 핵심 요소.

1. **마진 (Margin)**: 화면 좌우 맨 끝의 여백. 디자인에 가장 큰 영향을 주며, **그리드 설정 시 1순위로 정의**함. 마진을 제외한 영역이 실질적인 '컨테이너(Content Area)'가 됨. 이 영역을 쪼개서 그리드를 구성하는 게 추후 수정을 줄일 수 있음.
2. **컬럼 (Columns)**: 글자나 이미지 등 콘텐츠가 실제로 배치되는 세로 단. 브레이크 포인트에 따라 컬럼 개수나(일반적으로 모바일 4개, PC 12개) 폭을 조절함.
3. **거터 (Gutter)**: 여백 정의 역할. 컬럼과 컬럼 사이의 간격. 거터 수치에 따라 단의 폭이 자동으로 조정됨.

## 3. 레이아웃 그리드 종류 및 설정
피그마의 Layout Guide 패널에서 `+` 버튼을 누른 뒤, 세팅 아이콘을 클릭하여 아래 3가지 유형을 설정 가능.

### Grid (격자무늬)
가장 기본적인 모눈종이 형태의 가이드
- **Size**: 격자무늬 한 칸의 크기를 설정하는 영역. (기본 10px)
  - 10px를 32px로 바꾸면 칸의 크기가 커짐과 동시에 화면 내 격자의 개수도 자동으로 조절됨.
- **Color**: 그리드 선의 색상과 투명도(Opacity)를 설정하여 작업 시 눈의 피로도를 조절.

### Columns (세로 열)
좌측 상단 드롭다운에서 'Columns'를 선택하여 설정. 개수, 유형, 너비, 마진, 거터를 조합.
- **Count (개수)**: 화면을 나눌 단의 개수.
- **Type (유형)**: 컬럼의 위치와 크기 결정 방식입니다.
  - **Stretch**: 설정한 마진(Margin)과 거터(Gutter) 값을 바탕으로 컬럼 너비가 자동으로 변경되는 설정. (**Width = Auto**) 프레임 크기를 키우면 컬럼 너비도 같이 늘어나므로 반응형 레이아웃에서 주로 사용.
  - **Left / Right / Center**: 컬럼의 위치를 좌/우/중앙으로 지정. 이때는 컬럼의 너비(Width)를 수동으로 입력해야 함.

### Rows (가로 행)
컬럼의 '너비' 설정이 '높이(Height)' 설정으로 바뀐 형태.
- **활용**: 상단 앱바(App Bar)나 하단 탭바 등 가로 위계가 중요한 영역의 높이를 고정하거나 가이드를 잡을 때 유용.

## 4. 실무 활용 기술

### 그리드 중첩 (Multi-Grids)
하나의 프레임에 여러 그리드를 동시에 적용하여 복잡한 구조를 설계할 수 있음.
- **예시 (반응형 모바일 프로젝트)**: 
  1. **상단 앱바**: 일반적으로 고정으로 상시 노출. `Rows` (1개, 64px, Top)
  2. **하단 탭바**: 일반적으로 고정으로 상시 노출. `Rows` (1개, 고정 높이, Bottom)
  3. **콘텐츠(컨테이너) 영역**: 가변 콘텐츠 영역. `Columns` (4개, Margin 20, Gutter 16)
- **시각적 구분**: 각 그리드(Rows, Columns)의 **색상을 다르게 설정**하여 앱바 영역과 콘텐츠 영역을 직관적으로 구분.
- **그리드 복제**: 설정한 그리드를 선택하고 `Cmd` + `D` (또는 `Cmd` + `C/V`)를 눌러 빠르게 복사 가능.

### 스타일 등록 및 관리
자주 사용하는 그리드 조합은 스타일로 등록하여 효율적으로 관리.
- **Style 등록**: 레이아웃 가이드 패널의 `::` 아이콘 > `+` 버튼 클릭 > 이름 입력 후 `Create style`.
- **속성 및 범위**:
  - **로컬 파일 종속**: 생성된 스타일은 기본적으로 해당 파일 내 모든 프레임에서 자유롭게 재사용 가능.
  - **편집 (Edit)**: 등록된 스타일을 수정하면 해당 스타일이 적용된 모든 프레임에 실시간으로 반영.
  - **라이브러리(Library) 활용**: `Publish` 기능을 통해 등록된 스타일을 라이브러리에 올리면, 다른 파일에서도 해당 그리드 시스템을 빠르게 불러와 적용 가능.
- **연결 및 해제**:
  - **Detach Style**: 스타일과의 연결(상속)만 끊고, 현재 설정된 그리드 값은 그대로 유지하고 싶을 때 사용.
  - **-(Minus)**: 적용된 그리드 자체를 삭제.
